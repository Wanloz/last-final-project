**Выпускной проект**

+ Сфера деятельности – оператор связи.

+ Входные данные – информация о клиентах.

+ Задача – прогнозирование оттока клиентов из компании.

Результат - проведен исследовательский анализ данных Первоначально проведен исследовательский анализ данных, спланирован дальнейший план построения модели. В следующем этапе подготовлены данные, сгенерированы новые признаки. Выбраны модели машинного обучения: логистическая регрессия, решающее дерево, случайный лес, градиентный бустинг, многослойный перцептрон. Для каждой модели подобраны гиперпараметры рандомизированным поиском по сетке. Выбрал алгоритм градиентного бустинга CatBoost из всех обученных моделей. Метрики качества - AUC ROC. В качестве дополнительного задания проработаны способы удержания клиентов.

Библиотеки: pandas, numpy, seaborn, matplotlib, datetime, sklearn, xgboost, imblearn.over_sampling.
